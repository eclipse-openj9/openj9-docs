| **Tenure Age** | The number of times an object has been copied between allocate and survivor spaces (range 1-14). | <ul><li>[Garbage collection policies](gc.md)</li><li>[-Xgc](xgc.md)</li></ul> |
| **Tenure Area** | In gencon policy, the area of the heap where longer-lived objects are stored after surviving multiple scavenge cycles. | <ul><li>[Garbage collection policies](gc.md)</li><li>[Log examples](vgclog_examples.md)</li></ul> |
| **Thread Local Heap (TLH)** | A thread-specific memory cache that allows memory allocation without acquiring the heap lock, thus improving performance. | <ul><li>[Heap allocation](allocation.md)</li></ul> |
| **Tilting** | A technique that dynamically adjusts the boundary between allocate and survivor spaces based on survival rates. | <ul><li>[Garbage collection policies](gc.md)</li></ul> |
| **Tilt Ratio** | The proportion of the nursery area allocated to the allocate space versus the survivor space. | <ul><li>[Garbage collection policies](gc.md)</li></ul> |
| **Tracepoint** | A logging point in the VM code that records information about operations and exceptional conditions. See [`-Xtrace`](xtrace.md). | <ul><li>[-Xtrace](xtrace.md)</li><li>[New to Eclipse OpenJ9?](openj9_newuser.md)</li><li>[Trace formatter (traceformat)](tool_traceformat.md)</li></ul> |
| **Verbose GC Log** | A detailed log file containing GC metrics and operations, useful for performance analysis and tuning. See [Verbose GC logs](vgclog.md). | <ul><li>[Verbose garbage collection logs](vgclog.md)</li><li>[Garbage collection policies](gc.md)</li><li>[-Xverbosegclog](xverbosegclog.md)</li></ul> |
| **Very Hot Optimization** | A high JIT optimization level with profiling, used before scorching compilation to collect detailed profile data. See [Optimization levels](jit.md#optimization-levels). | <ul><li>[The JIT compiler](jit.md)</li><li>[-Xjit / -Xnojit](xjit.md)</li></ul> |
| **VM (Virtual Machine)** | The runtime environment that executes Java bytecode and manages application resources. | <ul><li>[New to Eclipse OpenJ9?](openj9_newuser.md)</li><li>[Garbage collection policies](gc.md)</li><li>[The JIT compiler](jit.md)</li></ul> |
