    -XXallowvmshutdown

    This option is provided as a workaround for applications that cannot shut down cleanly, as described in APAR IZ59734.

    Syntax

        -XX:allowvmshutdown:[false|true]
    Setting Effect Default
    false Disable
    true Enable yes
